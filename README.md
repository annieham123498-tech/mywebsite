## 🚀 빠른 시작

### 1. 저장소 클론

```bash
git clone https://github.com/annieham123498-tech/mywebsite.git
cd mywebsite
```

### 2. 의존성 설치

```bash
npm install
```

### 3. 로컬 개발 서버 실행

```bash
hugo server --disableFastRender
```

`http://localhost:1313` 에서 미리보기
